# Neo-Sense



-----------------pyFirmata 설치 방법-----------------
1. cmd 관리자모드로 실행
2. conda create -n prac(아무이름)
3. conda activate prac
4. pip install pyFirmata (설치)
5. Arduino IDE 실행
6. 보드 선택 후 업로드
7. File - Examples - Fymaata - StandardFymata
8. 업로드
9. 코드 실행


pyFirmata 안 깔릴 때

파이썬 코드와 패키지의 폴더가 같은지 확인
cmd에서 돌아가는 파이썬의 버전이 10인지 확인
아두이노 ide에서 아두이노에 업로드 두번 했는지 확인
